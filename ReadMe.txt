========================================================================
    Fortran Console Application : "VortoNeX" Project Overview
========================================================================

The Intel Fortran Console Application Wizard has created this "VortoNeX" project for you as a starting point.

This file contains a summary of what you will find in each of the files that make up your project.

VortoNeX.vfproj
    This is the main project file for Fortran projects generated using an Application Wizard.  It contains information 
	about the version of Intel Fortran that generated the file, and information about the platforms, configurations, and project 
	features selected with the Application Wizard.

VortoNeX.f90
    This is the main source file for the Fortran Console application. It contains the program entry point.
